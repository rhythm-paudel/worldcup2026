# World Cup 2026 Info App

A simple application that provides information and updates for the 2026 FIFA World Cup. The goal of this project is to make it easy for users to follow the tournament and access important information in one place.

## Features

* Live Scores
* Standings
* Teams
* Matches
* Tournament Bracket

## About

This project was created as part of my portfolio and focuses on delivering a clean and simple experience for following the 2026 FIFA World Cup.
